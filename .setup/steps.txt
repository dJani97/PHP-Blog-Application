___ setting up the development environment ___


start MySQL and execute the following:
	SET PASSWORD FOR root@localhost = PASSWORD('Password111');

	
edit config.inc.php:
	$cfg['Servers'][$i]['password'] = 'Password111';
	$cfg['Servers'][$i]['AllowNoPassword'] = false

	
edit httpd.conf and set it to something other than 80:
	Listen 8080
